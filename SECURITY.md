# Política de Segurança

## Versões Suportadas

Utilize sempre a versão mais recente para garantir que você tenha as correções de segurança mais atuais.

| Versão | Suportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reportando uma Vulnerabilidade

Levamos a segurança deste projeto a sério. Se você encontrar qualquer vulnerabilidade de segurança, por favor, **NÃO** abra uma issue pública.

Em vez disso, por favor, envie um e-mail para [mscarpenter.data@gmail.com].

Você deve receber uma resposta dentro de 48 horas. Se a vulnerabilidade for aceita, trabalharemos em uma correção e a lançaremos o mais rápido possível.

Agradecemos seus esforços para tornar este projeto mais seguro!
